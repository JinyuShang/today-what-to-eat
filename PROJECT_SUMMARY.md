## 🚀 如何部署

### 方法一：Vercel 网页部署（最简单）

1. **推送代码到 GitHub**
   ```bash
   cd today-what-to-eat
   git init
   git add .
   git commit -m "Initial commit"
   gh repo create today-what-to-eat --public --source=. --remote=origin
   git push -u origin main
   ```

2. **在 Vercel 部署**
   - 访问 https://vercel.com
   - 点击 "Import Project"
   - 选择 `today-what-to-eat` 仓库
   - 点击 "Deploy"
   - 等待完成（约 1 分钟）

### 方法二：Vercel CLI

```bash
cd today-what-to-eat
npm i -g vercel
vercel login
vercel
```
